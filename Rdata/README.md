# _Rdata_ folder

This folder contains R-data files with the raw observations and spatial data.

File _GIS.rda_ contains spatial data of the study area and camera locations, data on camera deployment and detection events for all target species.

Files in _occuRN_ sub-folder contain the fitted RN-models for each species along with results of goodness of fit test and model selection subsets and averages. These files were generated by the R script [/R/occuRN-models.R].
